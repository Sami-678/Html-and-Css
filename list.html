1. Ordered List

It describes the list of items written in order form

  1. Blue Berry
  2. Orange
  3. Mango
  4. Raspberry
  5. Apple

2. Unordered list

It describes the list in unordered or bullet point form

  • My name is Sami
  • I'm 22 years old
  • I like to play cricket
  • My favourite fruit is apple

3. Definition list

It describes the type of list which includes : dl(Defintion list) , dt(defintion term) , dd(definition description)
